How to Fill Out the Ticket Form

  1. Full Name: Enter your complete name (e.g., Jane Doe) in the "Full Name" input field.
  2. Email: Provide a valid email address where your ticket confirmation can be sent.
  3. Number of Tickets: Input how many tickets you want to purchase (e.g., 1, 3).
  4. Payment Status: Select whether you have paid. Choose Paid or Unpaid from the dropdown menu.
  5. Uploading a Text File: You can also upload a .txt file containing multiple entries. The file should include one user per line, formatted like:
    Jane Doe, jane@example.com, 2, Paid
    Then click the "Upload" button.
